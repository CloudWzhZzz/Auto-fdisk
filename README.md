#Auto fdisk
Auto fdisk for SpacePack Tools

##Demo

![Auto fdisk](https://dn-coding-net-production-pp.qbox.me/f65fdea1-c83a-41df-91c5-26b40b8a2f94.png) 

##How to use

```bash
$ wget https://mirrors.spacepack.org/tool/auto_fdisk.sh
$ chmod +x auto_fdisk.sh
$ ./auto_fdisk.sh
```
